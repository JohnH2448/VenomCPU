# VenomCPU
## RISC-V RV32I Pipelined CPU Core in SystemVerilog
### Specifications:
- In Order 5 Stage Pipeline
- Variable Latency MEM Interface
- M-Mode Privilaged ISA
- Base RV32I + Zicsr Support
